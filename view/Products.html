TerraExplorer Fusion跨平台網路三維地圖

基於 Web 的 3D 地理空間檢視器，可直接在Chrome、Edge、FireFox、Safari 等主流網頁瀏覽器中查看和分析高解析、極其逼真的 3D 場景，無需下載或安裝軟體。

產品優勢

全新的 TerraExplorer Fusion (TEF) 基於 TerraExplorer Desktop 源代碼轉換為 WebAssembly。此轉換技術提供了卓越的兼容性，使大多數原始 TerraExplorer Desktop 代碼能夠在網頁瀏覽器中按原樣運行，使得TEF具備以下優點 :

專業的引擎

新開發的 TerraExplorer Fusion 基於強大的單機版3D檢視軟體所使用的最先進的 3D 地理空間引擎。這個原生的 WebAssembly 引擎比以前的基於 Cesium 的引擎具有精進的功能。

性能改善

TerraExplorer Fusion 的 3D 地理空間引擎呈現全面的性能改進。這些包括更快的應用程序和內容加載時間、更順暢的幀速率和更好的記憶體管理，從而支援複雜的專案。

圖層和物件的支持

當發佈到 SkylineGlobe 服務器時，TEF 支援TerraExplorer單機版本所支援的絕大多數圖層和物件。

讀取相同的專案文件

TerraExplorer Desktop 和 TerraExplorer Fusion 都可以打開相同的 .fly 項目，提供兩個版本之間的無縫兼容性和互操作性。

統合API

TerraExplorer Desktop 可用的絕大多數 API 都可以在 TEF 中使用，確保外部工具和自定義之間的順利代碼共享。這使得您可以更輕鬆地利用現有資源並客製化 TerraExplorer Fusion 以滿足特定要求。

TerraExplorer Fusion 可以查看由 TerraExplorer Pro發佈的大多數線上圖層和物件，執行測量和分析操作，並根據任何客戶需求客製化網頁應用程序。TerraExplorer Fusion 的外觀樣式和功能可以輕鬆定制，以客製化所需的工具和功能。使用 TerraExplorer Fusion (TEF) 作為基礎，可以利用應用程序框架和功能，從而減少客製化方案所需的編程量。

伺服器私有雲共享完整的Web服務，用於發佈、存儲、管理和串流傳輸2D和3D地理空間資料，支援GIS資料類型包含3D網格、圖像、高程、向量和點雲。結合TerraExplorer Pro輕鬆一鍵發佈專案，SkylineGlobe Server 通過多種格式轉換以符合OGC規範發佈資料，讓您可以輕鬆共享數據，簡化工作流程，輕鬆打造遠端工作模式。

產品特性

多類型支援 3D 網格、影像、高程和點雲等GIS圖層類型，並提供發佈、存儲、管理和傳輸等 2D 和3D 空間數據服務，更可將數據輕鬆的共享。

圖片出處:skylinesoft.com

資料安全

完整帳號存取控制系統，可以輕鬆管理用戶或群組對資料的存取權限。 在單個服務器上託管多個虛擬站點，這些虛擬站點提供用戶層之間的資料與帳號完全分離，並允許個別站點的管理員帳號個別管理站點。

擴充性

支援叢集架構達到負載平衡，叢集中的所有服務器共享一個浮動授權，授權機制受限於授權的同時上線人數，不限安裝機器的數量，保有橫向擴充的彈性。

簡化工作流程

資料發佈可透過指定儲存裝置中的資料夾進行自動掃描發佈，亦可直接從 TerraExplorer Pro上傳並發佈單個地理空間圖層或完整專案到雲端服務器。通過單一的發佈操作，數據即可提供所有 TerraExplorer 用戶端、以及其他 2D 和 3D 用戶端使用。

集中式網頁管理

透過網頁介面進行服務管理，包含虛擬站點管理、資料來源管理，圖層管理、帳號及群組管理，資料使用流量統計與報表。

多項地形資料來源的發佈，完整且流暢的資料串流提升產出效率!

Terrain Services地形串流，能在發佈3D地形資料庫同時供數千位客戶使用，即時串流3D地形供遠端使用者瀏覽，提供3D Mesh、點雲、向量及圖像等資料。

資料互通性

Skyline用戶

SkylineGlobe Server以串流方式將2D和3D資料發送給TerraExplorer用戶。包括優化格式的所有3D網格、影像、高程和向量圖層、點雲資料，以及 TerraExplorer Pro 發佈的所有專案資料(模型、圖片等)。SkylineGlobe Server還提供了強大的搜索工具和管理搜索功能。
